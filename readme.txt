## External Service Usage
Our plugin relies on external services provided by TrackBee to function effectively. Specifically, we use the TrackBee API to setup the connection between the plugin and the TrackBee app. Once the connection is made the TrackBee API is used to track and analyze user behavior on your website.

- **TrackBee API Service:** [https://api.trackbee.ai](https://api.trackbee.ai) or [https://api.trackbee.io](https://api.trackbee.io)
- **Privacy Policy:** [TrackBee Privacy Policy](https://trackbee.io/privacy-policy)
- **Terms of Service:** [TrackBee Terms of Service](https://trackbee.io/terms-of-service)

Please review these documents to understand how your data is handled and to ensure compliance with legal requirements pertaining to data transmission and privacy.
